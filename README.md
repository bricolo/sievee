Sievee
============================================================
A c implementation to the sieve of erastosthenes.
Feel free to ask or comment *constructively*

## Installation
Download: `git clone https://github.com/bricolo/sievee`  
Then type `cd sievee`  
Then `make`  
This might build the `sievee` binary.
